# API VERSION 1

All paths are versioned with the version number (v#).

## GET
- /v#/auth/:username/:password
- /v#/categories
- /v#/categories/:category
- /v#/snippets
- /v#/snippets/category/:category
- /v#/snippets/category/:category/title/:title

## Post
- /v#/categories
- /v#/snippets

## Delete
- /v#/categories/:category
- /v#/snippets/category/:category/title/:title
- /v#/auth
