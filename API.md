# API VERSION 1

All paths are versioned with the version number (v#).

## GET
- /v#/auth/:username/:password
- /v#/categories
- /v#/categories/:key
- /v#/snippets
- /v#/snippets/category/:key
- /v#/snippets/:key

## Post
- /v#/categories
- /v#/snippets

## Delete
- /v#/categories/:key
- /v#/snippets/:key
- /v#/auth
